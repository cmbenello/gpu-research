## Implication

The 31m07s headline is tight at the granularity of run-to-run noise.
Further wins must come from algorithmic / architectural changes:
- GPUDirect Storage (cuFile + nvidia_fs kernel module) — would skip host
  pin entirely. Library is installed but `nvidia_fs` kernel module is not
  loaded on this host. Compat-mode falls back to host-buffered which would
  defeat the purpose.
- Concurrent partition+sort overlap (overlap 16m partition with 12m sort
  → projected 16-18m total).
- Multi-stream H2D pipelining within each per-bucket sort.
